# Churn Analysis in Gacha Game

## Project Overview

Project ini bertujuan untuk menganalisis faktor-faktor yang mempengaruhi churn (berhentinya pemain) dalam game gacha menggunakan data sintetis.

## Objective

- Mengidentifikasi faktor utama penyebab churn
- Memahami perilaku pemain berdasarkan data
- Menyiapkan dataset untuk analisis dan modeling

## Dataset

Dataset dibuat menggunakan simulator (data generator) yang meniru perilaku pemain.

Penjelasan lengkap dataset ada di `data_description.md`

## Analysis

Analisis dilakukan menggunakan:

- Exploratory Data Analysis (EDA)
- Perbandingan churn vs non-churn

## Key Questions

- Apakah session time berpengaruh terhadap churn?
- Apakah performa device mempengaruhi churn?
- Apakah hasil gacha mempengaruhi keputusan pemain?

## Files

- `dataset-generator` -> generate data utama
- `dataset.csv` -> data utama
- `analysis.ipynb` -> analisis
- `data_description.md` -> penjelasan dataset
